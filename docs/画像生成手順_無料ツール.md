# 画像生成手順（無料ツール使用）

## 概要
無料で使える画像生成AIツールを使って、「Infinite Rebirth Breaker」の画像アセットを生成する手順

---

## 推奨ツール

### 1. Bing Image Creator（最推奨）
- **URL**: https://www.bing.com/images/create
- **特徴**: DALL-E 3ベース、高品質、完全無料
- **制限**: 1日15-100枚程度（変動あり）

### 2. Leonardo.ai
- **URL**: https://leonardo.ai/
- **特徴**: ゲームアセット生成に特化、無料プランあり
- **制限**: 1日150枚程度

### 3. Stable Diffusion（Hugging Face Spaces）
- **URL**: https://huggingface.co/spaces
- **特徴**: 完全無料、オープンソース
- **制限**: なし（ただし、サーバー負荷により制限される場合あり）

---

## 生成手順

### Step 1: れいあキャラクターの生成

#### Bing Image Creator使用時
1. https://www.bing.com/images/create にアクセス
2. プロンプトを入力:
   ```
   A cute female software engineer character named Reia, anime style, chibi design, pink hair, wearing casual hoodie, sitting at computer desk, code symbols floating around, bright cheerful expression, high quality detailed illustration, white background
   ```
3. 「作成」をクリック
4. 生成された4枚から最適なものを選択
5. ダウンロード（PNG形式推奨）

#### Leonardo.ai使用時
1. https://leonardo.ai/ にログイン（無料アカウント作成）
2. 「Image Generation」を選択
3. モデル: `Leonardo Anime XL` を選択
4. アスペクト比: `1:1` を選択
5. プロンプトを入力（上記と同じ）
6. Negative Prompt: `blurry, low quality, distorted, watermark, text`
7. 「Generate」をクリック
8. 生成された画像から最適なものを選択してダウンロード

### Step 2: 表情バリエーションの生成

同じプロンプトに表情の指定を追加：

- **平常心**: `calm, relaxed expression, slight smile`
- **焦り**: `worried expression, sweat drop, slightly panicked`
- **パニック**: `panicked expression, wide eyes, flustered`
- **集中**: `determined expression, focused eyes, confident smile`

**重要**: 一貫性を保つため、基本プロンプトの最初に「same character design as previous image」を追加

### Step 3: タスクキャラクターの生成

#### バグタスク
```
A cute bug character, chibi style, red color scheme, glitch effect around it, error symbols (X marks), floating in the air, game sprite style, white background
```

#### 仕様変更タスク
```
A cute document character with question marks, chibi style, blue color scheme, paper/document design, confused expression, floating in the air, game sprite style, white background
```

#### レビュータスク
```
A cute clipboard character, chibi style, green color scheme, checkmark symbols, professional look, floating in the air, game sprite style, white background
```

#### 緊急タスク
```
A cute urgent task character, chibi style, orange/red color scheme, exclamation marks, alarm clock or warning symbols, floating in the air, game sprite style, white background
```

### Step 4: UI要素の生成

UI要素は、画像生成AIよりも**CSS/グラフィックツール**で作成する方が効率的です。

ただし、生成する場合は：
```
A horizontal progress bar, dark gray background, green to orange to red gradient fill, rounded corners, modern UI design, transparent background, simple and clean, game UI style
```

## 画像の後処理

### 1. 背景の透明化
- **ツール**: Remove.bg（無料版あり）または GIMP（完全無料）
- **手順**: 
  1. Remove.bg: https://www.remove.bg/ に画像をアップロード
  2. 自動で背景を削除
  3. PNG形式でダウンロード

### 2. サイズ調整
- **ツール**: GIMP または ImageMagick（コマンドライン）
- **推奨サイズ**:
  - キャラクター: 512x512px または 1024x1024px
  - タスク: 256x256px または 512x512px
  - UI要素: 256x256px または 512x512px

### 3. 最適化
- **ツール**: TinyPNG（無料版あり）または ImageOptim
- **目的**: ファイルサイズを削減してゲームの読み込み速度を向上

## コスト削減のコツ

1. **Bing Image Creator**: 1日の制限内で効率的に使用
2. **Leonardo.ai**: 無料プランの150枚を有効活用
3. **Stable Diffusion**: ローカル実行（GPU必要）で制限なし
4. **一貫性**: 同じプロンプトベースでバリエーション生成
5. **再利用**: 生成した画像を回転・反転・色調整でバリエーション作成

---

## トラブルシューティング

### 生成された画像が期待と違う場合
- プロンプトをより具体的に記述
- Negative promptで不要な要素を除外
- 生成回数を増やす（複数回試行）

### 一貫性が保てない場合
- 同じプロンプトベースを使用
- 「same style as previous image」を追加
- 生成した画像を参考画像として使用（Leonardo.aiのImage to Image機能）

### 背景が透明にならない場合
- Remove.bgを使用
- GIMPで手動で背景を削除
- プロンプトに「transparent background」を明記（効果は限定的）

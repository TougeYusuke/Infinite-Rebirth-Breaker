# 敵（タスク）画像のファイル名指定

## 必要な画像ファイル

以下の4種類のタスクタイプに対応する画像を準備してください。

### ファイル配置場所
```
public/assets/images/tasks/
```

### ファイル名一覧

| タスクタイプ | ファイル名 | 説明 |
|------------|----------|------|
| BUG（バグ） | `task_bug.png` | バグタスクの画像 |
| FEATURE（仕様変更） | `task_feature.png` | 仕様変更タスクの画像 |
| REVIEW（レビュー依頼） | `task_review.png` | レビュー依頼タスクの画像 |
| URGENT（緊急タスク） | `task_urgent.png` | 緊急タスクの画像 |

## 画像仕様

### 推奨サイズ
- **サイズ**: 120x120px または 128x128px（正方形推奨）
- **形式**: PNG（背景透過対応）
- **解像度**: 72dpi以上

### デザイン要件
- かわいらしいデザイン（れいあの世界観に合わせる）
- 各タスクタイプを視覚的に区別できるデザイン
- 背景透過（透明背景）
- シンプルで認識しやすいアイコン/キャラクター

### タスクタイプ別の特徴（参考）

#### BUG（バグ）
- 色: 柔らかい赤（#ff6b6b）
- アイコン: 🐛
- 特徴: 小さめ、かわいらしい

#### FEATURE（仕様変更）
- 色: 柔らかい青（#4ecdc4）
- アイコン: 📄
- 特徴: 中サイズ、落ち着いた印象

#### REVIEW（レビュー依頼）
- 色: 柔らかい黄（#ffe66d）
- アイコン: ✓
- 特徴: 中サイズ、優しい印象

#### URGENT（緊急タスク）
- 色: 赤（#ff4757）
- アイコン: ⚠
- 特徴: 大きめ、緊迫感のあるデザイン

## 実装予定

以下のファイルで画像読み込みと表示を実装します：
- `src/scenes/BootScene.ts`: 画像の読み込み
- `src/entities/Task.ts`: 画像の表示（現在のGraphics描画から画像スプライトに変更）
